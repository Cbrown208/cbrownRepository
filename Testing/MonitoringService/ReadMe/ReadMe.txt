To View the Monitor

Right click on the Project -> Debug - Start new instance

Start button (Windows Button) -> perfmon.exe -> Click on "Performance Monitor" 
Click the Green Plus Arrow - > Search for monitor name ex "MedassetsMonitoringTest"
Add all instances -> you should see an incrementor counting the logs as they come in 

Program will run for around 50 seconds. 

if you do not get there in time just re-debug the program and you will see the results.
